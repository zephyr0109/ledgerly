# 신한카드 지출 양식 변경 처리

- 실제 데이터는 b4부터 시작, 중간에 사용자별 중간소계가 있음
- 3얼 데이터 위치 : data/raw/202603/shinhancard_2603.xls
  - 엑셀로 변경한 파일 : data/raw/202603/shinhancard_2604.xlsx
- 데이터가 복합적으로 구성되어 있어서 엑셀의 구조 파악이 어려움. 잘 살펴보고 진행 필요
- 셀병합이 많이 되어있으며 대략적인 헤더 정보는

```
이용	이용	이용	이용	할	회	이번달 내실금액		적용	결제	포인트
일자	카드	가맹점	금액	부	차			구분	후	적립율
                기					잔액	(마이
                간						신한
                        원금	수수료			포인트)
                            (이자)
```

- 양식 부작용으로 신한카드에서 html파일을 굳이 xls로 지정하여 저장함. 이를 다시 xlsx로 변경한 상태
  - 즉 shinhancard_2603.xls 이파일은 html파일
  - xls 파일을 xlsx로 엑셀저장하는건 수동으로 하고 xlsx로 된 파일을 처리할지
  - 또는 xls 파일 자체를 읽어서 처리하는것도 자동으로 할지 판단 필요
