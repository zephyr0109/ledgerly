# Ledgerly

## 개요

이 프로젝트는 자산 관리 프로그램을 개발하는 토이 프로젝트입니다. 향후 들어갈 기능들은 다음과 같습니다.
- 지출 내역 저장 - 카드사 지출내역 엑셀파일 읽기 및 db 저장
- 지출 내역 카테고리별 분류
- 수입 대비 지출 내역 분석
- 자산, 부채 증감 추이
- 보고서 생성 및 웹 기반 UI 제공

## 개발 배경
현재 다수의 자산 현황 확인 앱은 매우 편리하게 되어있으나, 개인적으로 다음과 같은 문제가 있었습니다.
- 결혼한 이후 가족 전체의 자산을 보고싶은데 한계가 있음(카카오페이는 제공)
- 자산의 증가추이를 확인하고 싶으나 현재의 자산만 확인 가능(특정 앱은 제공하는 것 같기도)
- 위 두 문제점을 동시에 해결해 주는 앱은 없음

자산 관리를 엑셀로 매 월 수기로 관리하고 있었으나, 한 번 정산할 때 1,2시간 씩 걸리기 시작했고, 아이가 태어나면서 더이상 이렇게 진행하기 어렵겠다는 생각을 했습니다.

또한 개인적으로 연차가 쌓일 수록 실무와는 점점 동떨어지고 관리업무에 치중하다보니 개인적으로 개발할 시간이 줄어들기 시작했습니다.

그래서 스스로 연습할 겸 또 가계부 작성에도 도움을 받을 겸 이 프로젝트를 진행하게 되었습니다.
